You should have node, npm, angular-cli installed.
1. open command prompt and go to directory BOA.WeatherForcast\BOA.WeatherForcast.Web and run npm-install
2. run ng-build
3. to run test, run ng test on command prompt

4. To run application, open the solution using visual studio 2017.
5. Right click on the solution and choose, set startup projects and choose api and web projects as startup projects.
6  run the app using ctrl-f5.
   this will run both api and web application.
